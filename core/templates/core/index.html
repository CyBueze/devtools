DevTool API Documentation

DevTool is a simple API that helps you list, create, update and delete your favourite web development tools and resources.

Tools Used

  • Django and Django Rest Framework - For building the backend of the RESTful API
  • SwaggerUI - For proper documentation and testing of the RESTful API
  • Postgresql - SQLite was used even though Postgresql was demanded due to technical issues but it will be added in newer versions.
If you prefer to explore it using the drf interface, you can access the endpoints via the address bar:

/api/devtools - [GET request for listing all tools]

/api/devtools - [POST request for adding one tool]

/api/devtools/{id} - [GET request for listing a single tool]

/api/devtools/{id} - [PUT request for updating a single tool]

/api/devtools/{id} - [DELETE request for deleting a single tool]

Please Note

This current version of the project is incomplete but undergoing active development. Features that are currently absent but would be added in future versions include:

  • Postgresql will be used as a database to replace SQLite.
  • Authentication and authorization, using JWT and DRF permissions.
  • To access current version of this project, clone the GitHub repo by clicking https://github.com/CyBueze/devtools.git
